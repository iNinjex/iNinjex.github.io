# Sears-website-redevelopment-ideas This is a sears redevelopment idea from a course project in ITIS 4390/5390 we added dark mode toggle to out mobile/desktop version of our sears website. HTMLS: all htmls have new added dark mode toggle on the nav bar, whilst the mobile has it in the nav bar. script.js: This file makes it so that the dark mode toggle action works. when a user clicks on it, it goes straight to the dark mode and also changes its icon so that users can see that it has changed. style.css: this file holds code to handle the space, the colors and the overall set design of the dark mode. The site is set to change to a certain color throughout all of the pages so that everything will look flawless when navigating. ******************************** ADDED AND FIXED ********************************** DISCLAIMER : we used chatgpt to help debug issues suggest UI/UX improvements assist with JavaScript logic implementations (cart, search, loading states) clean up and merge code
